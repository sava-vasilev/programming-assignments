Arduino Safe

This project was from my first semester in Fontys and was for the subject ESO where we learned to work with Arduino and embedded systems in general.

The goal was to implement a working prototype of a Safe using only an Arduino Uno circuit board. The requiremets were that you can input a password, only the correct password can open the safe and an alarm had to go off whenever necessary. I managed to add another feature - being able to change the pre-set password. It might not sound like much but do keep in mind it had to be done with just 2 buttons and a pottentiometer (Knob).
